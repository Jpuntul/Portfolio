### Prerequisites

- Node.js (v14.0.0 or later)
- npm or yarn

### Installation

1. Clone the repository:

```
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

2. Install dependencies:

```
npm install
# or
yarn
```

3. Start the development server:
```
npm run dev
# or
yarn dev
```

4. Open http://localhost:5173/ in your browser.
